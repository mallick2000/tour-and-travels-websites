Got it! You want to build a React carousel with autoplay using the Embla Carousel + Autoplay plugin, just like in your example. Here’s a simple explanation and how you can use that logic to create a carousel that automatically scrolls through cards, stops when hovered, and resumes after.

Explanation of your code:

  • You create a ref for the Autoplay plugin instance with a 2-second delay.

  • You pass this plugin to the Carousel.

  • On mouse enter, autoplay stops.

  • On mouse leave, autoplay resumes.

  • You render 5 cards inside the carousel, each showing a number.

If you want to extend or customize this logic, here’s a slightly improved and annotated version:

jsx
"use client"

import * as React from "react"
import Autoplay from "embla-carousel-autoplay"

import { Card, CardContent } from "@/components/ui/card"
import {
  Carousel,
  CarouselContent,
  CarouselItem,
  CarouselNext,
  CarouselPrevious,
} from "@/components/ui/carousel"

export function CarouselPlugin() {
  // Create autoplay plugin with 2s delay; stop on user interaction (e.g., swipe)
  const autoplay = React.useRef(
    Autoplay({ delay: 2000, stopOnInteraction: true })
  )

  return (
    <Carousel
      plugins={[autoplay.current]}
      className="w-full max-w-xs"
      onMouseEnter={() => autoplay.current.stop()}
      onMouseLeave={() => autoplay.current.reset()}
    >
      <CarouselContent>
        {Array.from({ length: 5 }).map((_, index) => (
          <CarouselItem key={index}>
            <div className="p-1">
              <Card>
                <CardContent className="flex aspect-square items-center justify-center p-6">
                  <span className="text-4xl font-semibold">{index + 1}</span>
                </CardContent>
              </Card>
            </div>
          </CarouselItem>
        ))}
      </CarouselContent>
      <CarouselPrevious />
      <CarouselNext />
    </Carousel>
  )
}
